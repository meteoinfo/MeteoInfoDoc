************************
Streamline
************************

The streamlines will be traced and viewed according to the U/V components of the wind.

.. image:: ../../../_static/meteoinfo/grid_streamline.png

The streamline density could be adjusted from 1 to 10. The default value is 4. If change the 
value to 2, the streamlines look like following.

.. image:: ../../../_static/meteoinfo/streamline_density.png

.. image:: ../../../_static/meteoinfo/grid_streamline_density.png
