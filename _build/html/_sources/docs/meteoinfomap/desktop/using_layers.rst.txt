************************
Using Layers
************************

There are four kinds of layer in MeteoInfo.

- Vector layer (Point, Polyline, Polygon), has attribute data
- Image layer (geo-location setting, world file)
- Raster layer (Grid data)
- Web Map layer (Tile images)
