Show Pressure Profile¶

Click ‘Show Press Profile’ menu item, a ‘Press Profile Figure’ dialog will be opened. Press ‘Select Trajectory’ button (first one) in ‘Press Profile Figure’, and then select trajectory one by one in map view with mouse. The press profile line will be added when you select a trajectory. Please make sure ‘JuqanSpr’ layer was selected in legend, or the trajectories could not be selected. Trajectory could be removed from press profile figure by using ‘Remove Last Trajectory’ and ‘Remove All Trajectories’ button.
