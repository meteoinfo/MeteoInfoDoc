Output grid data¶

This tool will output PSCF or CWT grid layer to Surfer ASCII grid data file, which can be for further analysis such as contour fill plotting.

Press ‘Save to grid’ menu to open the following dialog.

Select a PSCF or CWT layer and a Field for output, and press save button to save the data to a Surfer ASCII grid data file. Then the file can be opened in Meteo Data dialog with ASCII Data -> Surfer ASCII Grid Data menu, and the data can be plotted as a contour fill layer.
