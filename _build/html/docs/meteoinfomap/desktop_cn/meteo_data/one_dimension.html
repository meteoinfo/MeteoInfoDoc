绘制一维图¶

对于多维的气象数据，只让其中一维可变，其他维都固定就可以取出一维数组绘制数据一维图。例如选择变量T，图形LINE_POINT，图形维Lon。可以 在维设置区域固定时间、高度和纬度维，经度维全取，绘制出来就是T变量沿经度变化的一维图。
