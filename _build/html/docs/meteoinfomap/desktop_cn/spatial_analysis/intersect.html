交集操作¶

交集操作（Intersect）是通过重叠处理得到两个图层的交集部分。例如有两个多边形图层，其中一个有4个空间要素，另一个有一个空间 要素，两个图层有重叠部分。点击“空间分析 -> 交集操作”菜单打开交集操作对话框，选择操作对象图层（Subject layer）和用做交集 操作的图层（Intersection layer），点击Apply按钮生成一个新的交集多边形图层。
