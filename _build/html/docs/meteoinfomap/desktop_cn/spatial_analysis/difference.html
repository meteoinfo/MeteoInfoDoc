差值分析¶

差值分析（Difference）是指通过重叠处理获得两个图层不同的部分。点击“空间分析 -> 差值”菜单打开差值分析对话框，选择操作对象 图层（Subject layer）和用做差值分析的图层（Difference layer），点击Apply按钮生成一个新的差值多边形图层 （Difference_test.shp）。
