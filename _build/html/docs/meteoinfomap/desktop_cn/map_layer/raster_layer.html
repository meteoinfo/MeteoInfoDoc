栅格图层¶

栅格图层包含了一个二维数组来表示格点数据以及相应的投影和坐标信息，图层是以图像的形式显示的。双击栅格图层名打开图层属性 对话框。栅格图层属性对话框中有图例（Legend）选项，可以对图例根据格点值进行设置，对于每个图例项的符号只能设置颜色和透 明度。
