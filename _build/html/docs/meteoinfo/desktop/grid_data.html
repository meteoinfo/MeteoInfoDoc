Plot Grid Data¶

Gridded data could be plot as following graphic types.

  • Contour

  • Shaded

  • Grid Fill

  • Grid Point

  • Vector

  • Barb

  • Streamline

  • Raster

The following examples were plotted with sample data ‘model.ctl’ and ‘model.dat’.

  • Contour and Shaded
  • Grid Fill and Grid Point
  • Raster
  • Vector
  • Barb
  • Streamline
