Attribute Table¶

The vector layer has attribute data. One row data in the attribute table corresponds with one shape of the layer. The attribute data of a shape which is highlighted could be identified using ‘Identifer’ tool button.

The shapes could be labeled by the attribute data with ‘Label’ tool.

The attribute data table could be opened by ‘Attribute Data’ menu.
