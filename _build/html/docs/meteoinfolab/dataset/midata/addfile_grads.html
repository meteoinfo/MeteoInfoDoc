addfile_grads¶

mipylib.dataset.midata.addfile_grads(fname, getfn=True)¶

Add a GrADS data file. use this function is GrADS control file has no .ctl suffix, otherwise use addfile function.

Parameters

  • fname – (string) GrADS control file name.

  • getfn – (string) If run __getfilename function or not. Default is True.

Returns

(DimDataFile) Opened file object.
