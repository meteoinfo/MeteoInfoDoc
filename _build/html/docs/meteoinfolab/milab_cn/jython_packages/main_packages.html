MeteoInfoLab的主要Jython包¶

MeteoInfoLab主要在底层Java代码的基础上开发了一系列科学计算和可视化的Jython包，包含在mipylib包中。其中最核心的包 是numeric，包含了多维数组的切片、索引和计算功能；plotlib包含了二维和三维绘图功能；dataset包含了科学数据读写的功能； geolib包含了一些GIS功能；meteolib包含了一些气象领域的分析算法；imagelib包含了一些图像处理功能；dataframe包含了 一些分析结构化数据的功能。通过这些Jython包可以完成气象科学数据的读写、分析计算、可视化的多种任务。
