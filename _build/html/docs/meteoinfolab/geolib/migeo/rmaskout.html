rmaskout¶

mipylib.geolib.migeo.rmaskout(data, x, y, mask)¶

Maskout data by polygons - the elements outside polygons will be removed

Parameters

  • data – (array_like) Array data for maskout.

  • x – (array_like) X coordinate array.

  • y – (array_like) Y coordinate array.

  • mask – (list) Polygon list as maskout borders.

Returns

(list) Maskouted data, x and y array list.
