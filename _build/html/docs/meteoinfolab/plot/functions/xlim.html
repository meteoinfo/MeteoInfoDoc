xlim¶

mipylib.plotlib.miplot.xlim(xmin, xmax)¶

Set the x limits of the current axes.

Parameters:

  • xmin – (float) Minimum limit of the x axis.

  • xmax – (float) Maximum limit of the x axis.

Example:

>>> xlim(2, 12)
