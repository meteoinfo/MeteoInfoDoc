ylim¶

mipylib.plotlib.miplot.ylim(ymin, ymax)¶

Set the y limits of the current axes.

Parameters:
  • xmin – (float) Minimum limit of the y axis.
  • xmax – (float) Maximum limit of the yaxis.

Example:

>>> ylim(2, 12)
