log10¶

mipylib.numeric.minum.log10(x)¶

Return the base 10 logarithm of the input array, element-wise.

Parameters:x – (array_like) Input values.
Returns:(array_like) The logarithm to the base 10 of x , element-wise.

Examples:

>>> log10([1e-15, -3.])
array([-15.,  NaN])
