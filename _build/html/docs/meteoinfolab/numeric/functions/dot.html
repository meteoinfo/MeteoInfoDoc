dot¶

mipylib.numeric.minum.dot(a, b)¶

Matrix multiplication.

Parameters

  • a – (2D Array) Matrix a.

  • b – (2D Array) Matrix b.

Returns

Result Matrix.

Examples:

>>> a = [[1, 0], [0, 1]]
>>> b = [[4, 1], [2, 2]]
>>> dot(a, b)
array([[4, 1]
      [2, 2]])
